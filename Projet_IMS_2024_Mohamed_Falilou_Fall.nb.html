Interpretation:

  • Le R_Squared : La proportion de la variance totale des donnes qui est explique par le modle1 est plus elev que ceux des 2 autres models. Plus le R-squared est lev, meilleur est lajustement du modle aux donnes. Donc le model1 est le meilleur model si nous nous basons uniquement sur le R_Squared.

  • Adjusted_R_squared : La valeure de lAdjusted R-squared du model1 indique un meilleur ajustement du modle aux donnes car etant plus eleve. Donc le model1 est le meilleur model si nous nous basons uniquement sur lAdjusted_R_squared.

  • Le model1 a la plus petite valeur dAIC (Akaike Information Criterion) donc il est le meilleur model.

  • Le BIC (Bayesian Information Criteria) du model1 est plus bas que celui des 2 autres donc nous choisisons le model1.

  • Le model1 a un variable prdictive Cylindre ayant un p-value significativement faible et est considr comme le meilleur car il indique une meilleure adquation du variable Cylindre utilise pour expliquer la variable Prix

# Dfinition de l'interface utilisateurs
ui <- fluidPage(
  titlePanel("APPLICATION DE PREDICTION DE PRIX DE VOITURES DE FABRICATION AMERICAINE SUR LE MARCHE DES ETATS-UNIS SELON LE NOMBRE DE CYLINDRES"),
  sidebarLayout(
    sidebarPanel(
      sliderInput("Cylindre", "Cylindre", 
                  min = min(num_vars$Cylindre), max = max(num_vars$Cylindre), 
                  value = median(num_vars$Cylindre))
    ),
    mainPanel(
      textOutput("prediction"),
      plotOutput("plot")
    )
  )
)

# Dfinition de la logique serveur
server <- function(input, output) {
  # Faire des prdictions bases sur l'entre de l'utilisateur
  output$prediction <- renderText({
    new_data <- data.frame(Cylindre = input$Cylindre)
    predicted_Prix <- predict(model1, new_data)
    paste("Le Prix prdit en Dollars est de", round(predicted_Prix, 2), "$")
  })
  
  # Crer un graphique de la rgression
  output$plot <- renderPlot({
    plot(num_vars$Cylindre, num_vars$Prix, 
         xlab = "Puissance du moteur", ylab = "Prix en $", 
         main = "Rgression linaire")
    abline(model1, col = "blue")
    points(input$Cylindre, predict(model1, data.frame(Cylindre = input$Cylindre)), col = "red", pch = 17)
  })
}

# Lancement de l'application
